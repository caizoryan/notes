# Mastering Emacs 

### Keys to remember

| Keybind   | Does                          |
|-----------|-------------------------------|
| C-x C-o   | open window in  cool          |
| C-c l = = | formats buffer                |
| M-d       | multi cursor match next       |
| M-D       | multi cursor quit             |
| M-u       | multi cursor undo last cursor |

### iBuffer filters

To filter buffers by directory just type / F and enter the project/folder name.

| Keys | Does                 |
|------|----------------------|
| / /  | Removes all filter   |
| / .  | filters by extension |
| / F  | filters by folder    |
